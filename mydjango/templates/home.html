A SIMPLE DJANGO PROJECT

KCTCS Educational Project

Core Technologies:

  • HTML5
  • Bootstrap/CSS
  • Python 3.12
  • Django 5.0.6
  • SQLite V.3

Project Description:

This project was designed for CIT 253 as a reference project for students to learn general structure and usage of the Python/Django web development environment. The project presents basic CRUD functionality with internal SQLite database that will follow the project and not require students to stand up server architecture in order to benefit from this resource.
